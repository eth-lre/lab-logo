## Miscellaneous

- The font is DIN Next, same as the ETH logo. The font does not need to be installed for the logo to display correctly.
- The used colors are:
  - black `#000000`
  - white `#ffffff`
  - pink: `#ffa7c0`
  - ETH red: `#b7352d`
  - ETH petrol: `#007894`
  - ETH gray: `#6f6f6f`
